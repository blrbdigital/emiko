# Emiko Golf — Site Map

## Overview
Premium custom golf head covers brand. Japanese-inspired streetwear-meets-golf aesthetic.
Logo: Leaping rabbit with Japanese text "バニー" (bunny) + "golf" script.

## Tech Stack
- **Platform**: WordPress (custom theme)
- **Hosting**: Hostinger (WordPress hosting)
- **Domain**: emiko.blrbdigital.com
- **Repo**: github.com/blrbdigital/emiko

## Design System
- **Background**: #0a0a0a (near-black)
- **Text**: #f5f5f0 (warm white)
- **Accent text**: rgba(255,255,255,0.3-0.7) for hierarchy
- **Borders**: rgba(255,255,255,0.06-0.08)
- **Fonts**: Inter (Latin), Noto Sans JP (Japanese)
- **Logo treatment**: filter: invert(1) to make black logo white on dark bg

## Sections (front-page.php)
1. **Hero** — Full-screen, logo, tagline "Your Game. Your Cover.", Japanese subtitle, CTA
2. **Marquee** — Scrolling ticker: leather, embroidery, handcrafted, etc.
3. **About** — Split layout: product-4 image + brand story + stats
4. **Gallery** — Feature row (2 large) + 6-item grid with hover overlays
5. **Lucky Collection** — Featured showcase of 6 colorful covers
6. **Process** — 4-step: Consultation → Design → Craft → Deliver
7. **Collections** — 3 cards: Personal, Corporate, Tournament
8. **Testimonials** — 2 quote cards
9. **CTA** — Email + Instagram buttons

## Deploy
- WordPress site on Hostinger — upload theme via Hostinger UI
- NOT using blrb-connect (WordPress, not static)

## Gotchas
- Logo is black-on-white JPG — uses CSS `filter: invert(1)` to appear white on dark bg
- All product images are PNG with white/transparent backgrounds
- product-4.png is 4.2MB (large) — may want to optimize before production
- Fonts loaded from Google Fonts CDN (Inter + Noto Sans JP)
